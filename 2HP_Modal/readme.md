# 2HPico Modal Resonator Sketch - very similar to Mutable Instruments Rings

**Dependencies:**

My fork of DaisySP library https://github.com/rheslip/DaisySP_Teensy

Adafruit Neopixel library

2HPicolib support library in this repository

**Usage:**

Top jack - trigger input

Middle jack - pitch CV input

Bottom jack - audio out

Page 1 parameters - Red LED

Pot 1 - structure

Pot 2 - damping

Pot 3 - brightness

Pot 4 - frequency

Page 2 parameters - Green LED

Pot 1 - accent

Pot 2 - sustain - rotate past 12 o'clock for sustained output

Pot 3 - unused

Pot 4 - unused
