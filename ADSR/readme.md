# 2HPico ADSR Sketch

Dependencies:

My fork of DaisySP library https://github.com/rheslip/DaisySP_Teensy

Adafruit Neopixel library

2HPicolib support library in this repository

Usage:

Top jack - gate input

Button - press for manual gate

Top pot - Attack time 0-2 seconds

Second pot - Decay time 0-2 seconds

Third pot - Sustain level 0-1

Fourth pot - Release time 0-2 seconds

Middle Jack - not used if module is configured for CV input, ADSR CV out 0-5V if configured for CV out

Bottom Jack - ADSR CV out 0-5V
