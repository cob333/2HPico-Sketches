# 2HPico Modal Resonator Sketch - very similar to Mutable Instruments Rings

**Dependencies:**

My fork of DaisySP library https://github.com/rheslip/DaisySP_Teensy

Adafruit Neopixel library

2HPicolib support library in this repository

***Use a Pico 2 board and overclock it to 250Mhz - this sketch is very CPU intensive***

**Usage:**

 top jack - trigger input
 middle jack - gate output
 bottom jack - pitch CV output

 page 1 parameters - Red LED
 Pot 1 - step 1 pitch
 pot 2 - step 2 pitch
 pot 3 - step 3 pitch
 pot 4 - step 4 pitch

 page 2 parameters - Orange LED
 Pot 1 - step 5 pitch
 pot 2 - step 6 pitch
 pot 3 - step 7 pitch
 pot 4 - step 8 pitch

 page 3 parameters GREEN LED
 Pot 1 - scale
 Pot 2 - clock divider
 Pot 3 - unused
 Pot 4 - overall pitch
